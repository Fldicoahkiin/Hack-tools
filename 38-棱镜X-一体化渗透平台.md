***\*0x02 安装与使用\****

一、Linux Amd64 运行示例：

```
$ wget https://github.com/yqcs/prismx/releases/download/1.0.10/prismx_linux_amd64.zip$ unzip prismx_linux_amd64.zip$ cd prismx_linux_amd64$ chmod +x prismx$ ./prismx
```

二、系统默认账号：

```
prismx/prismx@passw0rd
```
